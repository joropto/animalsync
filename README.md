# 🐾 AnimalSync - Smart Pet Collar

Open-source проект за интелигентен нашийник с фокус върху поверителността.

## 🛠️ Хардуер
- **Микроконтролер:** ESP32-S3 SuperMini
- **GPS:** GY-NEO6MV2
- **Биометрия:** MAX30102 (Пулс и O2)
- **Батерия:** 3.7V 4000mAh Li-Po
- **Корпус:** 3D принтиран (PLA/PETG)

## 🚀 Как работи?
Устройството следи локацията и здравето на домашния любимец в реално време, като предава данните директно към вашето устройство без външни облачни услуги.
